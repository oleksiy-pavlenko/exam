# BOOK01 CH01 Visual Assets

This directory contains app-ready visual assets for the current `BOOK01/CH01` scan batch.

- Manifest: [assets.json](assets.json)
- Full-page normalized images: `BOOK01-CH01-P001-normalized.jpg` through `BOOK01-CH01-P004-normalized.jpg`
- Figure and exercise crops: [crops](crops)

The JSON manifest is the app-facing source. It maps each image to a stable asset ID, source page ID, source line IDs, related concepts, related exercises, and intended app uses.
